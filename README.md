**🦦 GoLang Learning Journey 🚀**

This repository documents my journey of learning the Go programming language (also known as Golang) from various YouTube tutorials.

Lets' Go

Open your terminal or command prompt.
Navigate to the directory where your .go source file is located using the cd command.
Use the go run command in one of the following ways:

To run a specific file:
```bash
go run <filename>.go
```
Example: go run hello.go

To run all the code in the current directory (package main):
```bash
go run .
```
Example: If your current directory contains the source files for a main package, this command will run them
